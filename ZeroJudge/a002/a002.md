# 題目
a002. 簡易加法

# 解題思路
使用 `sys.stdin` 讀取輸入，並透過 `.split()` 將同一行中以空白分隔的兩個數字切開。接著使用 `int()` 將文字型態轉換為整數後進行相加，最後將兩數之和印出。

# 解題編號
18077480

# 程式碼
```python
import sys

for line in sys.stdin:
    if not line.strip():
        continue
    parts = line.split()
    a = int(parts[0])
    b = int(parts[1])
    print(a + b)
