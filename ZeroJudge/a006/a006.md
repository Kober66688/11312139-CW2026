# 題目
a006. 一元二次方程式

# 解題思路
讀取一元二次方程式的係數 $a, b, c$。利用公式判別式 $D = b^2 - 4ac$ 來決定根的情況：
1. 當 $D > 0$ 時，方程式具有兩個相異實根，公式為 $x_1 = (-b + \sqrt{D}) / (2a)$ 與 $x_2 = (-b - \sqrt{D}) / (2a)$，並依題目要求將大者排在前面。
2. 當 $D = 0$ 時，方程式具有重根，公式為 $x = -b / (2a)$。
3. 當 $D < 0$ 時，方程式無實根，直接輸出 "No real root"。

在程式中，利用 `math.isqrt(D)` 進行整數開根號，並使用整除運算子 `//` 確保結果輸出為整數型態。

# 解題編號
18077551

# 程式碼
```python
import sys
import math

for line in sys.stdin:
    if not line.strip():
        continue
    parts = line.split()
    a = int(parts[0])
    b = int(parts[1])
    c = int(parts[2])
    
    D = b**2 - 4*a*c
    
    if D > 0:
        sqrt_D = math.isqrt(D)
        x1 = (-b + sqrt_D) // (2 * a)
        x2 = (-b - sqrt_D) // (2 * a)
        print(f"Two different roots x1={x1} , x2={x2}")
    elif D == 0:
        x = -b // (2 * a)
        print(f"Two same roots x={x}")
    else:
        print("No real root")
