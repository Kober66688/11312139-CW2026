# 題目
a015. 矩陣的翻轉

# 解題思路
本題要求將一個 $R \times C$ 的二維矩陣進行轉置（將列與行互換），輸出一個 $C \times R$ 的全新矩陣。

由於測試檔案中包含多組矩陣資料，因此程式採用 `sys.stdin.read().split()` 一次性讀入所有以空白或換行分隔的元素，並透過迭代器 `iter()` 依序取出。每次讀取到矩陣的列數 $R$ 與行數 $C$ 後，便將後續的元素打包成二維清單。接著，利用 Python 內建的 `zip(*matrix)` 特性，將原本二維清單中各列的元素重新配對打包，輕鬆達到矩陣直橫對調（轉置）的效果。最後配合解包運算子 `*` 將翻轉後的數據逐列輸出。

# 解題編號
18087569

# 程式碼
```python
import sys

def solve():
    input_data = sys.stdin.read().split()
    if not input_data:
        return
        
    iterator = iter(input_data)
    
    while True:
        try:
            R = int(next(iterator))
            C = int(next(iterator))
            
            matrix = []
            for _ in range(R):
                row = [int(next(iterator)) for _ in range(C)]
                matrix.append(row)
                
            flipped_matrix = list(zip(*matrix))
            
            for row in flipped_matrix:
                print(*row)
                
        except StopIteration:
            break

if __name__ == '__main__':
    solve()
