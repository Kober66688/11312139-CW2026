# 題目
a017. 五則運算

# 解題思路
本題要求計算含有加、減、乘、除、取餘數及括號的四則運算式。由於涉及運算子優先順序（先乘除後加減）以及括號優先權，最嚴謹的做法是利用數據結構中的「雙堆疊法（Double Stack）」來實作表達式計算。

一個堆疊用來儲存操作數（數字），另一個用來儲存運算子。遍歷算式的每個字元：
1. 遇到數字直接推入數字堆疊。
2. 遇到左括號 `(` 直接推入運算子堆疊。
3. 遇到右括號 `)` 則不斷執行計算，直到彈出對應的左括號。
4. 遇到運算子時，若當前運算子的優先權小於或等於運算子堆疊頂端的運算子，則先將堆疊頂端的運算子彈出並計算，隨後再將當前運算子推入堆疊。

特別注意：題目要求除法運算為整數除法。在 Python 中，若直接使用 `//` 運算子在處理負數時會向下取整（例如 `-1 // 2` 為 `-1`），與 C/C++ 傳統向 0 靠攏截斷的定義不同，因此程式中使用 `int(a / b)` 來精準符合題目規範。

# 程式碼
```python
import sys

def precedence(op):
    if op in ('+', '-'): return 1
    if op in ('*', '/', '%'): return 2
    return 0

def apply_op(operators, values):
    if len(values) < 2: return
    b = values.pop()
    a = values.pop()
    op = operators.pop()
    if op == '+': values.append(a + b)
    elif op == '-': values.append(a - b)
    elif op == '*': values.append(a * b)
    elif op == '/': values.append(int(a / b))
    elif op == '%': values.append(a % b)

def evaluate(expression):
    for tokens in ['+', '-', '*', '/', '%', '(', ')']:
        expression = expression.replace(tokens, f' {tokens} ')
    tokens = expression.split()
    values = []
    operators = []
    for token in tokens:
        if token.isdigit() or (token.startswith('-') and token[1:].isdigit()):
            values.append(int(token))
        elif token == '(':
            operators.append(token)
        elif token == ')':
            while operators and operators[-1] != '(':
                apply_op(operators, values)
            if operators: operators.pop()
        elif token in ('+', '-', '*', '/', '%'):
            while operators and precedence(operators[-1]) >= precedence(token):
                apply_op(operators, values)
            operators.append(token)
    while operators:
        apply_op(operators, values)
    return values[0] if values else 0

def main():
    for line in sys.stdin:
        if not line.strip(): continue
        print(evaluate(line))

if __name__ == '__main__':
    main()
