# 題目
a003. 兩光法師占卜術

# 解題思路
使用 `sys.stdin` 讀取輸入的月份與日期，利用 `.split()` 將兩者分離並轉為整數型態。接著套用題目給定的公式 `(M * 2 + D) % 3` 算出結果 `S`。最後透過 `if-elif-else` 條件分支，根據 `S` 的值（0、1、2）分別輸出對應的運勢文字「普通」、「吉」或「大吉」。

# 解題編號
18077543

# 程式碼
```python
import sys

for line in sys.stdin:
    if not line.strip():
        continue
    parts = line.split()
    M = int(parts[0])
    D = int(parts[1])
    S = (M * 2 + D) % 3
    
    if S == 0:
        print("普通")
    elif S == 1:
        print("吉")
    elif S == 2:
        print("大吉")
