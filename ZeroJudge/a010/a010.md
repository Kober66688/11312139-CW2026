# 題目
a010. 因數分解

# 解題思路
模擬質因數分解的短除法流程。從最小的可能因數 `i = 2` 開始，若輸入的數字 `n` 能被 `i` 整除，則透過迴圈持續除以 `i` 並累加其出現次數（即次方的概念）。當無法再整除時，若次數為 1 則記錄 `i`，若大於 1 則記錄為 `i^次數`。為了提升執行效率，迴圈僅需執行至 `i * i > n`，若最後剩餘的 `n` 仍大於 1，代表剩餘的 `n` 亦為一個質因數，將其一併記錄。最後將所有記錄的因數以 `" * "` 連接字串後輸出。

# 解題編號
18077558

# 程式碼
```python
import sys

for line in sys.stdin:
    if not line.strip():
        continue
        
    n = int(line.strip())
    factors = []
    i = 2
    
    while i * i <= n:
        count = 0
        while n % i == 0:
            count += 1
            n //= i
        if count > 0:
            if count == 1:
                factors.append(str(i))
            else:
                factors.append(f"{i}^{count}")
        i += 1
        
    if n > 1:
        factors.append(str(n))
        
    print(" * ".join(factors))
