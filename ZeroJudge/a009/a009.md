# 題目
a009. 解碼器

# 解題思路
透過觀察範例輸入與輸出，發現密文字元與明文字元在 ASCII 碼上的差距為固定的常數 7（例如密文 `'1'` 的 ASCII 為 49，明文 `'*'` 的 ASCII 為 42）。因此，解題策略為使用 `sys.stdin` 逐行讀取密文字串，並走訪行內每個字元，利用 `ord()` 將字元轉為 ASCII 碼後減去 7，再透過 `chr()` 還原回明文字元。需要特別注意的是，每行結尾的換行符號 `\n` 應直接保留不進行轉換，以維持原本的文本結構。

# 解題編號
18077555

# 程式碼
```python
import sys

for line in sys.stdin:
    if not line:
        continue
        
    ans = []
    for char in line:
        if char == '\n':
            ans.append(char)
        else:
            new_char = chr(ord(char) - 7)
            ans.append(new_char)
            
    print("".join(ans), end="")
