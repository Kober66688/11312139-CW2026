# 題目
a004. 文文的求婚

# 解題思路
利用 `sys.stdin` 連續讀取多組年份輸入直到 EOF 結束。根據閏年的邏輯規則：「年份能被 4 整除且不能被 100 整除」或者「能被 400 整除」，在程式中使用 `if` 搭配 `and` 與 `or` 運算子進行判斷。符合條件者輸出「閏年」，否則輸出「平年」。

# 解題編號
18077546

# 程式碼
```python
import sys

for line in sys.stdin:
    if not line.strip():
        continue
    year = int(line.strip())
    if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
        print("閏年")
    else:
        print("平年")
